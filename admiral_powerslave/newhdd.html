More:
The Xbox hard drive is divided into many partitions, each representing a logical drive... "C:" and "E:" are the two primary drives on your Xbox drive. Drive F is the third primary partition which can be created when instlaling a larger hard drive. Drives X:, Y:, and Z: are all TEMPORARY drives, and mostly unimportant.  Do not worry if you do not see those when using FTP to transfer files to and from the Xbox. Drive D: will be the Xbox DVD drive.  The 3935 Version of the EvolutionX Dashboard supports Drive G: for larger than 137GB drives.

You cannot put a different HD into the Xbox without a modchip unless you have a means to lock the drive using the Xbox's hard drive password.  Unlocking your Xbox drive will make it unusable when the modchip is disabled or if you are using an 'exploit' to boot your system.

The Xbox hard drive uses a DIFFERENT file system called "FATX" that limits file and directory names to 42 characters in length and restricts the characters to a very specific set. This is why some games cannot be copied over to the Hard Drive without extensive patching.  They may contain characters the FATX will not allow to be writtem as part of the file name.

After you modded the xbox, for safety reasons, do not put a lot of things in E partition, this is your game saves partition.  If you are using a STOCK HDD for modding, don't put anything in it!

EDIT THE EVOX.INI FILE
Add to MENU:
Item "Install New drive",@212
Item "Install New drive with G",@213  [237+GB]

And add this to the bottom of the ini file:
[Action_12]

Info "Install HDD - F_Drive"
Warning "This will format all drives"

Progress "Formatting drives"

Format c:

Format f:
Format x:

Info "Install HDD - F & G_Drive"
Warning "This will format all drives"

When using @2XX, the XX is the [Action_XX] number for the routine.

Make your ISO with the files shown above, and burn with everything you need!!  You'll want to UNLOCK and BACKUP your Stock HDD before putting the new one in.  Everything from a stock Xbox HDD with nominal game saves, fits on one DVD.  This way you either can reuse it elsewhere, or keep it as a backup.  ALso, It is a good practice to unlock a drive before you remove it, to add another one.

When booting from the DVD, the main executable has to be "default.xbe" so if you are making a BOOT DISC from your version of  EvoX, then you need to rename "evoxdash.xbe" to "default.xbe" when making a boot disc.  I suggest copying your EvoX system to a seperate folder "EvoxBOOT" (Well, it fits), and using that as the source for the ISO.

What about heat issues for a larger HDD (7200RPM)?
Speed up the stock fan with the bios when editing it. The bios can be modified (using xbtool) to run the fan at any percent speed you want.  40% seems to be enough to compensate for the extra heat of a larger drive.

Max out the speed of the stock fan by cutting the black wire going from the fan to the motherboard.  You then splice in a wire for a, extra couple inches, so it can fastened under one of the motherboard screws (hard ground). This bypasses the bios speed setting, and runs the fan at 100% always.

OR, simply add extra fans to the sides of  the Xbox, they are vented.  The air is pulled in from the sides with the stock fan.
