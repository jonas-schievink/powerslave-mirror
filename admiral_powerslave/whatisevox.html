BASIC EXPLANATION OF EVOLUTION-X

Evolution-X (also known as EvoX) primary function is to turn your Xbox into a FTP server so that you can connect to it via a FTP client from your PC or MAC. 

The BIOS (256K) can be installed in to just about any mod chip.

EvoX (The DASHBOARD) also replaces your existing dashboard (GUI) when the Mod chip with an EvoX Bios is activated.  This replacement dashboard allows you to run any "unsigned" code on the Xbox, like Emulators, DVD Player Program, Utilities and more.

Evolution-X  the Dashboard  is different from Evolution-X the bios.

EvoX Dshboard is illegal because it is built with the XDK, which is Microsoft's Development software, and is licensed to them only.

You can play games directly off of your harddisk using EvoX (And others)..

You will be able to install a larger HDD.

EvoX 3935 now supports TRAINERS (.ETM)

Other Dashboards also allow you do so the same.
