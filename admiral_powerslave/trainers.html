TRAINERS

Trainers allow you to cheat in games by patching memory with custom code values. 

For instanmce, we know (so they say at Bungie) there are no CHEAT CODES for HALO or HALO II, but there is a TRAINER for Halo II.  This will allow Infinate shield, Infinate ammo with no reoload, rapid fire mode, No 1 hit Sniper kills, and Invincibility. 

Trainers do the same thing as cheat codes in most cases, except the game loads with them already on, so to speak.  In fact, they are called TRAINERS because when developiong a game, they are used so the developer can go through a complete "walk through" of the game to be sure it works, and is free of bugs.

IF a trainer says "HaloII +13" that means there are 13 different cheats/options to activate in the trainer.

So far, Trainers work automatically with the EvolutionX 3935 revision dashboard.   As long as you have the trainers stored as C:\TRAINERS on the Xbox HDD, it will show up on the Dashboard.   You cannot use these trainers with an Action Replay! They are not save games, and  no one ever said they were! SO that means you can't  load them to your memory card.. They, again, are NOT saves.

Usually, you can not use NTSC trainers with PAL games and vice versa.  Some trainers do support both, and they will indicate that with "NTSC/PAL."  Setting your Xbox region to a PAL region does not change the game from NTSC to PAL. It changes the video mode (which can screw you bigtime), and not the code of the game.  If you are using a PAL copy of a game with a NTSC trainer nothing will happen (in regards to the trainer) if you switch your video mode to NTSC.

You can not use trainers on Xbox Live, no those cheaters are NOT using trainers, so leave it at that!

A trainer file ends in .ETM - any other file in an Archive or with the trainer does NOT have to go into the Trainers folder.

You can edit the trainer with EvoX, turn on options, then turn on the Trainer, then Exit and Save.  When you boot the game from the EvoX Dashboard, the trainer is used for that game. 

Will This Permanently Hack The Game?  No, Just that one time.
