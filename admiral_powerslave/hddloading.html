PLACING GAMES / DVDS ON XBOX'S HARD DISK

#1 - You must have a Modded or Softmodded Box!

1. Obtain DVD2XBOX or PX_HDLOADER
2. Copy the program into E:\APPS [Stock HDD]
        Copy the program into F:\APPS [Upgraded HDD]
3. Reboot to non M$ Dashboard
4. Go to where you Launch Apps - You should see it listed
5. Launch that App
6. Follow instructions, menus, and prompts.

Example directory structure should be:
F:\APPS\HDLOADER\[files] - default.xbe being the main executable.

With HD_LOADER, you may have to launch that app in some cases to run the games.  It will create a lilst of games for you at the menu.   This is the program I have been using, but I added this folder to the game menu.

DVD2XBOX is supposed to be a better program.  It uses "ACL" (Auto Correction Lists) to patch games as they are copied to the HDD.

A Non upgraded HARD DISK will not hold many games at all, maybe two at most! 

If a game fails to run after loading it to the HDD, then you need to 1st: Delete the content of the CACHE drives, X,Y, and Z.   If that does not work, then you have to delete the game save and start over.   This has fixed 60% or more of these failing problems.



IF running a game and you get the Dirty Disc Error that means there was a file that was either not found, or currupted, or unsigned (like a map).
