 What is an emulator?
An emulator is a program which mimics other operating systems, or other systems such as game consoles. Emulators try to translate or interpret the original program code of the original game and try to make it work on your Xbox. Games for emulators of cartridge based game consoles are usually stored in ROM files. Games for emulators of CD based game consoles like the Playstation however use the original game CD.
What is a ROM file?
Most of the original classic games are stored on game cartridges. For example the Atari 2600, Nintendo NES and Sega Genesis (among many others) use a cartridge. The games are stored on those cartridges using ROM chips (Read Only Memory). Some people have created special devices which can read the data from the ROM chips, and can save that data as a ROM file. In other words, a ROM file contains all data from the game cartridge. Of course for systems which don't use game cartridges, such as the Playstation, ROM files don't exist (except for needing the BIOS). In case of the Playstation you can put the Playstation CD in your CD-ROM drive to play games on a Playstation emulator.

        {
            AutoAddItem "e:\games\"
