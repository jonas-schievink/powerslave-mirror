This is the interesting part. First mode select returns 'ERR'. Second mode select probably sends a random data block to the drive to encrypt. Drive encrypts this block, sends it back to the xbox, via mode sense (page code 3B), xbox checks if the block was correctly encrypted. If so, it knows that the drive has the same key as the xbox has. Then it starts looping 03/00 commands to see if a disc is inserted. This will keep looping till a disc is inserted. If a disc is inserted, xbox sends a mode sense with pagecode 3E (= authentication) and will start the authentication process, pretty much the same as we saw in the original xbox.
