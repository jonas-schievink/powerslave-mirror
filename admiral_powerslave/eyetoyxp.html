USE EyeToy CAMERA ON WIN XP

Firstly flip your EyeToy over and look at the bottom of the stand, you will need to check the Sony Part ID, so you can use the appropriate Product ID code for the drivers.

Download the appropriate pre-edited inf for your EyeToy

 
"Fixed" Inf Files
    Download
SLEH-00031 PID_0155 Inf
SLEH-00030 PID_0154 Inf

Unzip the correct driver archive to a folder.

Plug the EyeToy into a USB port - Will say "EyeToy USB camera Natami".

 The new hardware installation wizard wil popup on the screen.

Select the Install from list or specific location (Advanced) option then click next.

Select "Search for the best driver in these locations"

Check Include this location in the search: click browse and point it to where you extracted the edited drivers then click next.

Windows will go off searching for your drivers.

If successful it will change to "D-Link VGA WebCam", and continue the installation.

You will be presented with a prompt screen, about it has not passed the logo testing to verify ots compatibility with Windows XP, just select "Continue Anyway".

Next a small box titled "sel3110" with an exclamation point, will popup on your screen press "OK".

You will now be looking at the configuration screen for the drivers, for 50Hz (which is PAL) or 60hz (which NTSC).  It asks of you live in a 50Hz area.  If you are NTSC  (USA), select "NO".

Installation is complete for the VIDEO side.  The AUDIO Device will be detecetd next, and will (should) install without any user intervention.

You can also do a replace the words "D-Link VGA Webcam" in the OV519.INF file with any title you would like "Sony PS2 EyeToy" for example. This will change the name of the device during installation only.
