When EvoX boots, all you get is a default skin, with the list of games, no other menu options. 

There are probably more things you can remove from the file, but, this is what gave me JUST a list of games on the screen to choose from. 

This small ini gives you network support and other functions.  If you do not need ANY of this, go HERE

file.  IF you ONLY want a MULTI GAME BOOT DISC, then go below for the SIMPLEST evox.ini file.
THE SIMPLEST WAY:

Make an "evox.ini" file with just this text:

[Menu]

         Section "Games"
        {
            AutoAddItem "d:\games\"
            SortAll
        }

YEP, that's it!

1. Open NOTE PAD
2. Highlight, and COPY the text in YELLOW
3. PASTE into Notepad
4. Save file as "evox.ini"

Make your disc!

NO network, No Skin, no nothing, just a menu with games, that is ALL you need!

Works with APPS as well.
