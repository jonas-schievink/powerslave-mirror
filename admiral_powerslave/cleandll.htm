CLEAN SYSTEM DIRECTORY - v1.6
FreeWare!

This program scans your system (EXEs and other DLLs), looking for all references to DLL files that reside in your Windows system directory (C:\WINDOWS\SYSTEM). Those DLL files in the system directory that have no programs calling on them can be moved out of the system directory, saving disk space and improving system performance. For experienced users only!

Requirements: Windows 3.1 or Windows 95

The DLLs found that are not called by any other programs, or DLLs, are moved to a backup directory. You should leave those files there for a few months, and make sure all your current software runs without errors. If you are satisfied that removing them will not harm any installed applications, then remove them. You may want to BACK them up first, then delete them.
