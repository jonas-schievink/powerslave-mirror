THE SWITCHED MOD CHIP:

You will need a Double Poll, Single Throw (DPST) switch for the following operation.  If you can find one, a MOMENTARY "ON" switch will do.  READ ALL BEFORE ATTEMPTING!
This version shows the MOD-CHIP outside the PSX (For easy replacement)
IF your MOD CHIP is inside, you can run wires 5, and 6 to the center POLLs of of the DPST witch.
Then, from the other two leads, run wires to correct points on PSX main-board.
Mount switch where you can toggle it.
When you BOOT a backup game (that has ANTI-MOD measures), make sure the switch is ON.  As soon as the WHITE screen turns to the BLACK with the PSX logo, turn it off immediately!  If you have a momentary ON switch, hold the button IN (on) until the white screen disappears, then release (off).
You might be able to simply use a SINGLE poll, SINGLE throw stitch to turn the power ON and OFF to the chip, using PIN ONE (Vdc).  Pin EIGHT is ground.  Pins 5 and 6 send the correct data to the sony CPU that tells the PSX there are bad blocks, and the country code is correct. Have the power ON when the PSX is on, then turn the power to the chip OFF as soon as the white screen goes black with the PSX logo.  LEAVE the chip off till you change games, then turn the power to the PSX off, turn the mod ON, then repeat the proceedure.
NOTE: Any modifications will void the PSX warranty!
Also, the reason the game can detect the old MOD-CHIP is because it is always ON, sending the information along the bus.  STEALTH-MOD-CHIPS are supposed to turn themselves off after the process, but some games can catch it.  Newer PLAYSTATION, SPCH 7000's and above, also have an Anti-Mod code built into the ROM of the PSX.  This will detect the MOD at power up, and will shame you for having one in your PSX.  The STEALTH-MOD works in timing issues, and it's still not perfected.
