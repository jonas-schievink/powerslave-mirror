STREAMING MEDIA WITH XBOX MEDIA CENTER [XBMC]
  Audio - Video - Pics



Download and install "ccxgui_1.3" from http://www.xboxmediacenter.com/download_files.htm
This program takes about 6K of memory, and can be run as "NT Service"  Very small application~

Install it on your PC & Run. The GUI should pop up.

You add shares by clicking on "Add Share" button or drag and drop from
Windows Explorer. You'll want to add directories that contain the files
you want to share, like Audio, Video, and Pictures, Example:

Click on START or RESTART server button. IT will tell you if the server
is running or not. IT displays your PC's IP address, and PORT 1400 if running.

Turn on XBOX, and start Xbox Media Center. WE are going to use AUDIO
files as an example.

GO to My Music - Press A.

Go down to "XBMSP Network Servers (Auto-Discover)" - Press A

You should get (top of screen) Server: 192.168.X.X (PcName) - Press A

Your SHARED folders should display!!

Note: When you pic My Audio, only AUDIO files will play. If you go to a
Video or Picture folder, and press "A", nothing will display in it.

Once on the folder that has your MP3, or other compatible files, press "A"
to go into the folder. Then select file to play -press "A".

You are DONE, and enjoy your streams from your PC.
